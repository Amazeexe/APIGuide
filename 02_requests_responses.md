
### API Requests and Responses
 APIs use the HTTP protocol to handle requests and responses.  
- **HTTP Methods:**  
  - `GET`: Retrieves data.  
  - `POST`: Sends data.  
  - `PUT`: Updates data.  
  - `DELETE`: Removes data.
